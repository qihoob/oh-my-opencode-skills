# Skill: dev-context-first

**角色**: 开发 (Dev)
**功能**: 开发前获取最小上下文
**触发关键词**: 给我上下文、先看看代码、先了解下

## 核心能力

1. **最小上下文原则**
   - 只获取完成任务所需的最低限度信息
   - 避免过早优化和过度设计
   - 快速启动开发

2. **上下文内容**
   - 相关文件列表
   - 现有代码结构
   - 关键函数/类位置

3. **按需获取**
   - 根据任务复杂度调整上下文范围
   - 小任务 → 单文件
   - 大任务 → 模块级

## 使用场景

```typescript
// 示例 1: 简单修改
"修改登录页面的提交按钮样式"

// 示例 2: 功能实现
"实现用户头像上传功能"

// 示例 3: Bug 修复
"修复首页加载慢的问题"
```

## 工具可用
- glob, grep, read, lsp_symbols
