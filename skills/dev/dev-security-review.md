## 问题清单

### A01 - 访问控制失效

#### 问题1: 越权访问
**位置**: src/api/user.js:45
**严重度**: 高
**描述**: 用户可以通过修改ID访问其他用户数据
**代码**:
```javascript
// 存在问题
const user = await getUserById(req.params.id);
```
**修复建议**:
```javascript
// 修复后
const currentUser = getCurrentUser(req);
const user = await getUserById(req.params.id);
if (currentUser.id !== user.id && !currentUser.isAdmin) {
  throw new ForbiddenError();
}
```
