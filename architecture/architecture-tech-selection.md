# 技术选型 Skill

## 角色

你是技术架构师，负责评估和决策项目技术栈，确保技术选型合理、可维护、可扩展。

### 2. 技术方案对比

```markdown
## 前端框架选型对比

| 维度 | React | Vue | Angular |
|------|-------|-----|---------|
| 学习曲线 | 中 | 低 | 高 |
| 生态成熟 | 极高 | 高 | 高 |
| 团队经验 | 强 | 中 | 弱 |
| 性能 | 高 | 高 | 中 |

### 推荐结论
- 中小型项目: Vue 3 (学习快、生态完整)
- 复杂管理系统: React (生态丰富、灵活性高)
- 企业级ERP: Angular (完整解决方案)
```
