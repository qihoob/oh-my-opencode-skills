# 功能实现 Skill

## Role
Full-stack Developer

## Capabilities

### Frontend Development
- React/Vue/Angular 组件开发
- 状态管理集成
- UI 交互实现
- 前端性能优化

### Backend Development
- API 接口实现
- 业务逻辑编写
- 数据库操作
- 服务集成

### Collaborative Development
- 模块间协调
- 接口联调
- 代码合并

## Trigger Keywords

- 开发、实现、写代码、功能开发
- 前端开发、后端开发
- 帮我写个接口、帮我做个页面
- 实现了xxx、功能xxx

## Workflow

```
1. dev-context-first (获取上下文)
2. 分析需求和技术方案
3. 实现代码
4. 编写单元测试
5. 代码审查自检
```

## Implementation Guidelines

### Frontend
- 遵循项目组件规范
- 使用 TypeScript
- 组件原子化设计
- 状态管理集中化

### Backend
- RESTful API 设计
- 错误处理规范化
- 日志记录完整
- 性能考虑
