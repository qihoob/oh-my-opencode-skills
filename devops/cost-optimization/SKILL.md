# Skill: cost-optimization

**角色**: DevOps / 技术负责人
**功能**: 成本优化与 FinOps
**触发关键词**: 成本优化、云成本、FinOps、预算管理、资源优化、降本增效

## 产出文档
**路径**: `.opencode/docs/cost-optimization-{date}.md`
**说明**: 成本分析报告和优化建议

## 依赖文档
- 云资源清单
- 成本账单数据

## 核心能力

1. **成本分析**
   - 按服务/部门/项目分摊
   - 历史趋势分析
   - 异常支出检测
   - 预算对比

2. **资源优化**
   - 闲置资源识别
   - 资源配置调整
   - 预留实例建议
   - 自动扩缩容优化

3. **架构优化**
   - Serverless 替代方案
   - 存储分层策略
   - CDN 优化
   - 数据传输优化

## 输出格式

```markdown
# 成本优化报告

## 成本概览

| 类别 | 本月 | 上月 | 变化 |
|------|------|------|------|
| 计算 | $5,000 | $4,500 | +11% |
| 存储 | $1,000 | $1,000 | 0% |
| 网络 | $500 | $600 | -17% |
| **总计** | **$6,500** | **$6,100** | **+7%** |

## 成本分布

### 按服务
| 服务 | 成本 | 占比 |
|------|------|------|
| EC2 | $3,500 | 54% |
| RDS | $1,500 | 23% |
| S3 | $800 | 12% |
| 其他 | $700 | 11% |

## 优化机会

### 高优先级 (预计节省 $1,000/月)

#### 1. 闲置 EC2 实例清理
- **资源**: i-0abc123 (t3.large)
- **状态**: CPU 使用率 < 5% 持续 30 天
- **建议**: 停止或降级
- **预计节省**: $50/月

#### 2. RDS 实例降配
- **资源**: prod-db (db.r5.large)
- **现状**: 平均 CPU 15%，内存 30%
- **建议**: 降至 db.t3.medium
- **预计节省**: $200/月

#### 3. S3 存储分层
- **对象**: 10TB 日志文件
- **现状**: Standard 存储
- **建议**: 转 Infrequent Access
- **预计节省**: $150/月

### 中优先级 (预计节省 $500/月)

#### 4. 预留实例购买
- **服务**: EC2 长期运行实例
- **建议**: 购买 1 年预留实例
- **预计节省**: 40% 按需价格

#### 5. NAT Gateway 优化
- **现状**: 每个 AZ 一个 NAT Gateway
- **建议**: 使用 NAT Instance 替代
- **预计节省**: $100/月

## 行动计划

| 优化项 | 负责人 | 截止日期 | 预计节省 |
|--------|--------|----------|----------|
| 闲置实例清理 | DevOps | 本周五 | $50/月 |
| RDS 降配 | DBA | 下周一 | $200/月 |
| S3 分层 | DevOps | 下周五 | $150/月 |

## 监控指标

- 月度成本预算：$7,000
- 优化后预计：$5,000
- 成本警报阈值：$6,500 (93%)
```

## 优化检查清单

### 计算资源
- [ ] 清理闲置实例
- [ ] 合理选择实例类型
- [ ] 使用自动扩缩容
- [ ] 考虑预留实例
- [ ] 使用 Spot 实例（适用于无状态）

### 存储
- [ ] 实施存储分层
- [ ] 清理过期数据
- [ ] 启用生命周期策略
- [ ] 压缩和去重

### 网络
- [ ] 优化 CDN 配置
- [ ] 减少跨区域传输
- [ ] 使用私有连接
- [ ] 压缩传输数据

### 数据库
- [ ] 合理配置实例
- [ ] 实施读写分离
- [ ] 使用连接池
- [ ] 定期清理数据
