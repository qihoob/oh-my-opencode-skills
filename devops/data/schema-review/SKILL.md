## 核心能力

### 1. 表结构合理性检查

```
检查维度:

| 检查项 | 检查方法 | 通过标准 |
|--------|----------|----------|
| 字段命名 | 是否遵循命名规范 | 小写下划线，含义清晰 |
| 类型选择 | 是否选择了合适的类型 | 金额用DECIMAL，布尔不用CHAR |
| NOT NULL | 是否合理设置 | 业务必填字段 NOT NULL |
| 默认值 | 是否设置了合理默认值 | 状态字段有初始值 |
| 字段冗余 | 是否存在不必要的冗余 | 冗余字段有明确理由 |
| 审计字段 | 是否包含审计字段 | created_at, updated_at |
| 主键策略 | 主键类型是否合适 | BIGINT自增或UUID |
| 外键约束 | 是否合理使用外键 | 评估性能 vs 一致性 |
| 软删除 | 是否需要软删除 | 有 deleted_at 字段 |
```

### 2. 索引合理性审查

```
审查清单:

缺失索引检测:
| 场景 | 检测方法 | 建议 |
|------|----------|------|
| WHERE 条件字段无索引 | 分析查询模式 | 添加普通索引 |
| JOIN 关联字段无索引 | 检查外键字段 | 添加关联索引 |
| ORDER BY 字段无索引 | 分析排序字段 | 添加排序索引 |
| GROUP BY 字段无索引 | 分析分组字段 | 添加联合索引 |

冗余索引检测:
| 场景 | 检测方法 | 建议 |
|------|----------|------|
| 单列被联合索引覆盖 | idx(a,b) 存在时 idx(a) 冗余 | 删除单列索引 |
| 重复索引 | 两个索引完全相同 | 保留一个 |
| 未使用索引 | 查询计划中未使用 | 评估是否删除 |

索引选择性评估:
| 选择性 | 基数/行数比 | 建议 |
|--------|-------------|------|
| > 0.9 | 高选择性 | 适合建索引 |
| 0.1 - 0.9 | 中选择性 | 考虑联合索引 |
| < 0.1 | 低选择性 | 不适合单独索引 |
```

### 3. 安全合规检查

```
安全审查:

| 检查项 | 说明 | 处理方式 |
|--------|------|----------|
| PII 字段 | 姓名、手机号、邮箱 | 加密存储或脱敏 |
| 密码字段 | 密码、密钥 | 哈希存储，禁止明文 |
| 敏感数据 | 身份证、银行卡 | 加密 + 访问控制 |
| 日志表 | 是否记录了敏感信息 | 脱敏处理 |
| 审计追踪 | 关键操作是否有记录 | 添加审计字段或审计表 |
| 数据保留 | 数据过期清理策略 | 定义保留周期 |
```

### 4. 性能风险评估

```
风险评估矩阵:

| 风险项 | 评估方法 | 风险等级 |
|--------|----------|----------|
| 全表扫描 | 缺少索引的查询 | 高 |
| 大表 DDL | ALTER TABLE 锁表风险 | 高 |
| N+1 查询 | 关联查询模式 | 中 |
| 大事务 | 事务范围过大 | 中 |
| 热点更新 | 高并发更新同一行 | 高 |
| 死锁风险 | 多表交叉更新 | 中 |

大表变更评估:
| 表行数 | DDL 方式 | 建议 |
|--------|----------|------|
| < 10万 | 直接 ALTER | 标准操作 |
| 10万-100万 | 在线DDL | 监控锁等待 |
| 100万-1000万 | pt-osc / gh-ost | 使用在线变更工具 |
| > 1000万 | 分批变更 | 评估停机窗口 |
```

### 5. 与契约一致性检查

```
契约对齐验证:

| 检查项 | 说明 |
|--------|------|
| 数据模型字段覆盖 | 契约中定义的每个字段在表中都有对应列 |
| 类型映射正确 | 契约类型到数据库类型的映射是否正确 |
| 枚举值一致 | 状态字段枚举值与契约一致 |
| 约束一致 | NOT NULL / UNIQUE 与契约约束一致 |
| 无幽灵字段 | 表中没有契约之外的字段 |
```

### 6. 与代码一致性检查

```
ORM Model 对比:

| 检查项 | 检查方法 |
|--------|----------|
| 字段名映射 | Model 属性名 vs 数据库列名 |
| 类型映射 | Model 类型 vs 数据库列类型 |
| 关联关系 | Model 关联 vs 数据库外键 |
| 索引定义 | Model 索引注解 vs 数据库索引 |
```
