# 协同 Skills

跨角色协作的 Skills，整合所有交接、评审、复盘流程。

## 目录结构

```
collaboration/
├── handoff/          # 角色间交接
│   ├── product-to-dev.md     # 产品→开发
│   ├── dev-to-qa.md           # 开发→测试
│   ├── visual-to-product.md   # 视觉→产品
│   └── visual-to-dev.md       # 视觉→开发
├── review/           # 评审活动
│   ├── acceptance.md          # 验收评审
│   └── code-review.md        # 代码审查
├── process/          # 流程管理
│   ├── retrospective.md      # 迭代复盘
│   ├── iteration-closure.md   # 迭代闭环
│   └── bug-coordinator.md    # Bug协调
└── sync/             # 同步对齐
    ├── context-alignment.md  # 上下文对齐
    └── incident.md           # 故障响应
```

## 技能列表

### 1. 交接 (handoff)

| Skill | 说明 | 触发词 |
|-------|------|--------|
| `collab-product-to-dev` | 产品→开发需求交接 | 转开发、交给开发 |
| `collab-dev-to-qa` | 开发→测试交接 | 提测、转测试 |
| `collab-visual-to-product` | 视觉PM与产品PM协同 | 设计评审、视觉确认 |
| `collab-visual-to-dev` | 视觉PM与开发交接 | 设计稿交接、视觉验收 |

### 2. 评审 (review)

| Skill | 说明 | 触发词 |
|-------|------|--------|
| `collab-acceptance-review` | 验收评审、UAT | 验收、UAT |
| `dev-code-review` | 代码审查 | code review、审查 |

### 3. 流程 (process)

| Skill | 说明 | 触发词 |
|-------|------|--------|
| `collab-retrospective` | 迭代复盘 | 复盘、retro |
| `collab-iteration-closure` | 迭代闭环 | 迭代闭环 |
| `collab-bug-coordinator` | Bug协调 | Bug修复 |

### 4. 同步 (sync)

| Skill | 说明 | 触发词 |
|-------|------|--------|
| `collab-context-alignment` | 上下文对齐 | 对齐、确认范围 |
| `collab-incident` | 故障响应 | 故障、应急 |
