# Skill: collab-product-to-dev

**角色**: 协同 (Collaboration)
**功能**: 产品到开发的需求交接
**触发关键词**: 需求交接、需求评审、开发确认

## 核心能力

1. **需求澄清**
   - 详细说明功能需求
   - 解答开发疑问
   - 提供参考示例

2. **技术对齐**
   - 确认技术实现方案
   - 识别技术风险
   - 讨论替代方案

3. **交接确认**
   - 明确完成标准
   - 约定后续沟通机制
   - 记录决策要点

## 输出格式

```markdown
## 需求交接单

### 功能名称
[名称]

### 需求描述
[详细描述]

### 验收标准
1. 标准 1
2. 标准 2

### 技术注意事项
- 注意事项 1
- 注意事项 2

### 依赖模块
- 模块 A
- 模块 B

### 预计工作量
[X] 人天
```

## 工具可用
- read, write, grep
