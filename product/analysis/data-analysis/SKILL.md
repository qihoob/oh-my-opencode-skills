# Skill: data-analysis

**角色**: 产品 (Product)
**功能**: 数据分析与埋点设计
**触发关键词**: 数据分析、埋点、指标设计、数据追踪、转化漏斗

## 产出文档
**路径**: `.opencode/docs/data-tracking-{feature}.md`
**说明**: 数据埋点设计文档，包含指标定义、埋点位置、分析维度

## 依赖文档
- **必须读取**: `.opencode/docs/requirement-{feature}.md`（需求文档）`

## 核心能力

1. **指标体系设计**
   - 北极星指标 (North Star Metric)
   - 一级指标 (业务核心指标)
   - 二级指标 (功能级指标)
   - 过程指标 (用户行为指标)

2. **埋点规划**
   - 页面浏览 (PV/UV)
   - 点击事件
   - 转化漏斗
   - 异常事件

3. **分析维度**
   - 用户分群
   - 时间维度
   - 渠道来源
   - 设备/地域

## 输入要求

用户需提供：
- 功能需求文档
- 业务目标（如：提升转化率）

## 分析维度

- **用户维度**: 新用户/老用户、会员等级
- **时间维度**: 小时/日/周/月
- **渠道维度**: 来源渠道、推广活动

## SQL 示例

```sql
-- 日活计算
SELECT COUNT(DISTINCT user_id) as dau
FROM events
WHERE event_name = 'page_view'
  AND DATE(event_time) = CURRENT_DATE
```
