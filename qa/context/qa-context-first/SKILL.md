# Skill: qa-context-first

**角色**: 测试 (QA)
**功能**: 测试前获取最小上下文
**触发关键词**: 测试上下文、了解被测功能、测试范围

## 核心能力

1. **功能理解**
   - 理解被测功能
   - 识别关键业务流程
   - 了解用户场景

2. **测试范围**
   - 确定测试边界
   - 识别依赖模块
   - 明确测试重点

3. **环境信息**
   - 测试环境配置
   - 测试数据准备
   - 账号权限需求

## 工具可用
- read, grep, glob
